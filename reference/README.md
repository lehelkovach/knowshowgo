# Reference Implementation

This directory contains reference implementations and test files from the Python version of KnowShowGo.

## Contents

### Python Reference Implementation

- `python/knowshowgo.py` - Core KnowShowGo API (Python)
- `python/ksg_orm.py` - ORM implementation (Python)
- `python/models.py` - Data models (Python)
- `python/tests/` - Python test suite

## Purpose

These files serve as:
1. **Reference implementation** - Shows how features are implemented in Python
2. **API compatibility guide** - Ensures JavaScript API matches Python API
3. **Test migration reference** - Python tests guide JavaScript test development
4. **Design documentation** - Real code examples of the design

## Using the Reference

When implementing JavaScript features:
1. Check the Python implementation for the design pattern
2. Ensure the JavaScript API matches the Python API signature
3. Use Python tests as a guide for JavaScript tests
4. Maintain semantic compatibility between implementations

## Note

These files are included for reference only. The JavaScript implementation may differ in implementation details while maintaining API compatibility.
